# market-news-helper

# 📈 시황 챌린지 뉴스 헬퍼 (Market Commentary Assistant)

실시간 뉴스 검색, AI 분석, 시황 코멘트 작성 및 구글 스프레드시트 연동을 한 번에 해결하는 웹 애플리케이션입니다.

---

## 🌟 주요 기능

1. **실시간 뉴스 검색 및 AI 시황 분석**
   - 구글 검색 그라운딩(Google Search Grounding) 도구가 적용된 **Gemini API**를 통해 실시간 최신 뉴스를 탐색합니다.
   - 탐색한 실시간 뉴스를 분석하여 **뉴스 핵심 요약**, **시장 영향도/분위기(Bullish / Bearish / Neutral)**, **AI 시황 분석 & 투자자 전망**을 카드 형태로 시각화합니다.
   - 요청 한도 초과(429) 감지 시 자동으로 지수 백오프(Exponential Backoff) 기반 재시도를 수행하여 오류를 최소화합니다.

2. **키워드 즐겨찾기 (Favorites)**
   - 자주 분석하는 핵심 키워드들을 태그 칩으로 등록하여 원클릭으로 검색창에 입력할 수 있습니다.
   - 사용자가 직접 키워드를 추가(`+`)하거나 불필요한 키워드를 제거(`x`)할 수 있으며, 이 설정은 웹 브라우저의 `localStorage`에 자동 저장 및 유지됩니다.

3. **코멘트 템플릿 및 자동 불러오기**
   - 시장 핵심 화두, 주요 변수, 종합 전망 등 자주 쓰는 코멘트 시작구 템플릿을 한 번의 클릭으로 편리하게 삽입합니다.
   - **어제 코멘트 불러오기** 기능을 통해 정확히 어제(오늘 - 1일) 작성했던 코멘트 내용을 즉시 로드하고, 오늘 날짜로 자동 변환하여 시황 챌린지 작성 과정을 획기적으로 줄여줍니다.

4. **코멘트 작성 히스토리 목록 (최대 30개)**
   - 구글 시트로 전송에 성공한 모든 로그(날짜, 키워드, 코멘트 등)는 로컬 브라우저 히스토리에 자동 누적됩니다.
   - 헤더의 시계 아이콘(히스토리 목록)을 클릭해 과거 작성했던 내용을 확인하고 원클릭으로 다시 복원할 수 있습니다.

5. **구글 스프레드시트(Google Sheets) 즉시 연동**
   - 전송 버튼 클릭 시, 구글 앱스 스크립트(Google Apps Script) 웹 앱으로 데이터를 발송하여 구글 시트의 지정된 시트에 날짜, 키워드, AI 분석 요약문, 사용자 코멘트가 자동으로 예쁘게 정렬되어 추가됩니다.

---

## 🛠 기술 스택

- **Frontend**: Vanilla HTML5, JavaScript (ES6), CSS3
- **Design & Themes**: 프리미엄 글래스모피즘(Glassmorphism) 다크 모드, 네온 그래디언트 포인트 컬러, 스켈레톤 로딩 애니메이션
- **AI Engine**: Gemini 2.0 Flash (`google_search` grounding enabled)
- **External Integration**: Google Apps Script Web App API

## 🚀 시작하기 & 사용 방법

1. 본 레포지토리의 파일들(`index.html`, `style.css`, `app.js`)을 로컬 서버나 브라우저를 통해 실행합니다.
   - 로컬 서버 실행 예시 (Python 3): `python -m http.server 8000`
2. **구글 스프레드시트 및 Gemini 연동 설정 (필수)**:
   - 연동할 구글 시트에서 `확장 프로그램 > Apps Script`를 실행합니다.
   - 워크스페이스 내 `google-sheets-script.js` 안의 코드를 복사하여 기존 스크립트에 덮어쓰고 저장합니다.
   - Apps Script 에디터 좌측 **프로젝트 설정(톱니바퀴 아이콘)**을 클릭합니다.
   - **스크립트 속성 편집**을 누르고, 속성명 `GEMINI_API_KEY`를 추가한 뒤 값에 사용자의 Gemini API Key(AQ. 또는 AIza 형식)를 넣고 저장합니다.
   - 우측 상단 `배포 > 새 배포`를 눌러 **웹 앱** 유형으로, 실행 권한은 **나(Me)**, 액세스는 **모든 사용자(Anyone)**로 설정하여 배포합니다.
   - 생성된 **웹 앱 URL**을 복사합니다.
3. 웹 페이지 우측 상단의 **톱니바퀴(설정) 아이콘**을 클릭합니다.
   - **Google Sheets Web App URL**: 복사한 웹 앱 URL을 입력하고 설정을 저장합니다. (보안을 위해 API 키는 웹 브라우저가 직접 다루지 않고 Apps Script를 통해서만 안전하게 연동됩니다.)
4. 원하는 시황 키워드를 검색하여 분석을 진행하고, 코멘트를 작성한 뒤 **Google 시트로 전송하기**를 누릅니다!

## 🛠 문제 해결 (Troubleshooting)

### 1. Gemini API 에러 (401 또는 403) 발생 시
- Google AI Studio의 인증 정책 개편으로 인해 발급받은 API 키의 권한이나 형태(`AQ.` 또는 `AIza`)에 따라 인증 실패가 날 수 있습니다.
- 이 경우:
  1. Apps Script의 스크립트 속성(`Script Properties`)에 입력한 `GEMINI_API_KEY` 값이 정확한지(앞뒤 공백 등 포함 여부) 더블 체크해 주세요.
  2. 에러가 지속되면 [Google AI Studio](https://aistudio.google.com/)에서 새로운 API 키를 재발급받아 `GEMINI_API_KEY` 속성값을 갱신 후 배포 버전을 업데이트해 보세요.
  3. 여전히 오류가 해결되지 않을 경우, 브라우저 개발자 도구(F12) 콘솔 탭에서 구체적인 API 에러 원문 메시지를 확인하실 수 있습니다.
