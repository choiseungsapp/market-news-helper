애플리케이션 설정

AI 분석 및 데이터 기록을 담당하는 구글 스프레드시트 배포 웹 앱 주소입니다.
구글 스프레드시트 및 Gemini 연동 방법:
  1. 구글 스프레드시트에서 확장 프로그램 > Apps Script를 실행합니다.
  2. google-sheets-script.js 코드를 복사하여 기존 스크립트에 덮어쓰고 저장합니다.
  3. 좌측의 프로젝트 설정(톱니바퀴 아이콘)을 클릭합니다.
  4. 스크립트 속성 편집을 누르고, 속성명 GEMINI_API_KEY를 추가한 뒤 값에 사용자의 Gemini API Key(AQ. 또는 AIza 형식)를 넣고 저장합니다.
  5. 우측 상단 배포 > 새 배포를 눌러 웹 앱 유형으로, 실행 권한은 나(Me), 액세스는 모든 사용자(Anyone)로 설정하여 배포합니다.
  6. 생성된 웹 앱 URL을 위 입력칸에 입력하고 저장합니다.
